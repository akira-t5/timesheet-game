📹 タイムシート計算とは

タイムシート計算は、アニメーション制作で使われる技術です。アニメーターが描いた原画を何フレーム表示するかを計算する作業になります。

🎬 フレーム（コマ）について
  • 1秒 = 24フレーム（コマ）
  • 例：3秒 = 72フレーム
  • 例：1秒12コマ = 24 + 12 = 36フレーム
🎮 ゲームの流れ
  • 表示されたフレーム数を「秒 + コマ」に変換します
  • 例：72フレーム → 3秒 + 0コマ
  • 例：85フレーム → 3秒 + 13コマ
  • 計算して秒とコマを入力し「判定する」をクリック
  • 正解すると次の問題へ進みます
⚡ 難易度説明
  • 入門 (〜3秒)：最大 72フレーム / 初心者向け
  • 簡単 (〜6秒)：最大 144フレーム / 基礎習得向け
  • 中級 (〜12秒)：最大 288フレーム / 中級者向け
  • 上級 (最大)：最大 999フレーム / 上級者向け
🎯 計算ポイント
  • 秒 = フレーム数 ÷ 24（整数部分）
  • コマ = フレーム数 ÷ 24（余り）
  • コマは 0〜23 の範囲です
🏆 スコア計算
  • 正解数：全問題中何問正解したか
  • 合計時間：全問題に要した時間
  • 平均時間：1問あたりの平均時間
  • 最も速い平均時間が最高記録になります
💾 記録機能について
📊 自動保存について
  • ゲームを終了すると、自動的に結果が保存されます
  • 保存操作は不要です
  • ブラウザを閉じても記録は保持されます
🎖️ 記録される情報
  • 難易度（入門・簡単・中級・上級）
  • 正解数 / 総問題数
  • 合計時間と平均時間
  • プレイ日時
  • 各問題の正解・不正解と回答時間
🆕 新記録通知

以下の場合、結果画面に新記録のバナーが表示されます：

  • その難易度で初めてプレイした時
  • 正解数が前回より増えた時
  • 平均時間が前回より短くなった時
📈 「過去の記録を見る」機能
  • 全プレイ履歴を新しい順で表示
  • 難易度別の最高記録を表示
  • 最高正解数と最速平均時間を確認可能
  • 日時付きで詳細情報を確認できます
💿 データ保存場所

このゲームは以下の方法で自動的にデータを保存します：

  • localStorage：ブラウザのローカルストレージ（優先）
  • IndexedDB：より大容量のデータベース（代替）
  • サーバーには送信されません（プライベート）
  • ブラウザのキャッシュクリアで削除されます
🔄 記録のリセット
  • 「過去の記録を見る」画面の「記録をリセット」で削除可能
  • 確認ダイアログが表示されます
  • 削除後は元に戻せないため注意してください
🌐 GitHub公開版での動作
  • ✅ GitHub Pages：完全対応

  • ✅ raw.githubusercontent.com：完全対応

❓ よくある質問
  • Q: 複数デバイスで同期される？
    A: されません。各デバイスのブラウザに独立して保存されます。
  • Q: シークレットモードで記録される？
    A: ブラウザによっては記録されない場合があります。
  • Q: データは安全？
    A: ローカルのみに保存され、サーバーには送信されません。
